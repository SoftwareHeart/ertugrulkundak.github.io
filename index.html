Merhaba! Ben Ertuğrul Kundak, Full Stack Yazılım Mühendisiyim.

Finansal teknoloji sektöründe 2+ yıl deneyime sahip olarak
modern teknolojiler kullanarak ölçeklenebilir ve performanslı
çözümler geliştiriyorum.

ASP.NET Core, Entity Framework ve React teknolojileri ile
mobil ve web uygulamaları için güvenli RESTful API'ler
tasarlıyorum.

Portföy sayfası performansını %40 artırdım ve 30+ grafik içeren
finansal analytics dashboard'ları geliştirdim.

KNN algoritması ile yapay zeka destekli fon öneri sistemi
tasarladım. JWT tabanlı kimlik doğrulama, middleware geliştirme
ve real-time data visualization konularında deneyim sahibiyim.

Uzmanlık Alanlarım:
- ASP.NET Core & .NET Framework
- RESTful API Geliştirme
- Machine Learning & KNN Algorithm
- Financial Data Analytics
- Performance Optimization
- Database Design & Optimization

32 takımın katıldığı hackathonda 2. sıra elde ettim.
